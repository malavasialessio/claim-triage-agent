Conclusions

What we built. What we learned.

A production-ready triage pipeline

From raw email to routed ticket in seconds. Operator review built in from day one — not as an afterthought.

Self-improving without retraining

Few-shot injection from operator corrections: seconds to activate, fully reversible, zero infrastructure. The agent got better while we were presenting.

Deterministic safety that survives prompt changes

PreToolUse hook blocks emergency routing unconditionally. A guardrail that lives outside the model context window cannot be bypassed by prompt injection.

What Claude Code enabled
  • Hooks — deterministic guardrails, not probabilistic prompts
  • /start · /stop — runbooks as repeatable slash commands
  • CLAUDE.md — shared conventions enforced from session 1
  • Coordinator pattern — 5 specialist tools, full audit trail

Honest gaps

Customer history is mocked. Adversarial set covers 20 cases — not enough for production. The self-improving loop is logically correct but needs more real-world data to prove the accuracy trend empirically. Baseline: 80% golden accuracy, 5% false-confidence rate.
